# ComfyUI-Foundation1

Custom nodes for the Foundation-1 audio model in ComfyUI.

## Features
- **Foundation-1 Prompt Builder**: Structured tag-based prompting for music generation.
- **Foundation-1 KSampler**: BPM-aware sampler with automatic duration sync and internal latent handling.
- **Audio-to-Audio**: Seamless transformation of existing audio signals.
